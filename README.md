SketchWire
==========

Sketch in 3D
-----

Homepage
-----

#[SketchWire](http://robincwillis.github.io/SketchWire/)

Requirements
-----
- Processing Core Library 1.x
- OpenGL
- PeasyCam


Interaction
-----
- Left Click : Draw
- Space : Toggle Camera/Drawing Mouse Control
- D : Rotate Right
- A : Rotate Left
- W : Rotate Up
- S : Rotate Down
- R : Zoom In
- F : Zoom Out
